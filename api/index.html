CNPJ Data API Documentation

Welcome to the documentation for the CNPJ Data API. This API allows you to retrieve company details based on a CNPJ (Cadastro Nacional da Pessoa Jurídica) number.

API Endpoint

The API endpoint for retrieving company details is:

http://{url}/cnpj/{cnpj}

Replace {cnpj} with the CNPJ number you want to look up.

Usage

To use the API, you can make a GET request to the endpoint using any HTTP client or browser. For example, using the fetch API in JavaScript:

fetch('http://{url}/cnpj/{cnpj}')
      .then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error('Error:', error));

Response

The API returns company details in JSON format. Here is an example response:

[
      {
        "cnpj": "00000000",
        "razao_social": "BANCO DO BRASIL SA",
        "natureza_juridica": "2038",
        "qualificacao_responsavel": "10",
        "capital_social": 120000000000,
        "porte_empresa": "05",
        "ente_federativo_responsavel": ""
      }
    ]

Error Handling

If the provided CNPJ is not found or there is an error in the request, the API will return an empty array or an error message, respectively.
